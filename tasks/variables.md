# Variables

Replace text in <...> to make the code to the values specific for your project.

1. ek_ecommerce_events - Looker project and its model file name. We recommend for a Looker project and its model file (if there is one) to have the same name.
    * update the name of the model file with the same ek_ecommerce_events.
    * update connection name in the model file
2. BigQuery - database name where the data is located and to which is the connection in the model file point to.
3. `prj-s-dlp-dq-sandbox-0b3c.badal_dq.ecommerce_events` - table name that is going to define the 1st view
4. ecommerce_events - view name tha that you want your 1st view to have
5. <table_name_2> - table name that is going to define the 2nd view
6. <view_name_2> - view name tha that you want your 2nd view to have
7. <explore_name_1> - explore name for your 1st explore
8. <datagroup_name_1> - datagroup name
